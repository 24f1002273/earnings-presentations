Example: NPV Calculation Service

Internal analytics microservice used for project and portfolio evaluation: 


def npv(rate, cash_flows):
    """
    Simple NPV calculation.
    :param rate: discount rate as decimal (e.g. 0.08)
    :param cash_flows: list of cash flows (CF_0, CF_1, ..., CF_T)
    """
    return sum(cf / (1 + rate) ** t for t, cf in enumerate(cash_flows))

Core Financial Metrics

Net Present Value (NPV):

\[ \text{NPV} = \sum_{t=0}^{T} \frac{CF_t}{(1 + r)^t} \]

Return on Equity (ROE):

\[ \text{ROE} = \frac{\text{Net Income}}{\text{Shareholders' Equity}} \]

Cost-to-Income Ratio (CIR):

\[ \text{CIR} = \frac{\text{Operating Expenses}}{\text{Operating Income}} \]

Briefly define each metric: NPV for investment decisions, ROE for shareholder returns, and CIR for operational efficiency. Tie back to the quarter's numbers.
