Financial Model Implementation

Python code for earnings per share calculation:

def calculate_eps(net_income, shares_outstanding):
    """
    Calculate Earnings Per Share
    
    Args:
        net_income: Net income in millions
        shares_outstanding: Shares outstanding in millions
    
    Returns:
        float: EPS value
    """
    eps = net_income / shares_outstanding
    return round(eps, 2)

# Q4 Financial Data
net_income = 418  # million
shares_outstanding = 120.5  # million
diluted_shares = 124.2  # million

basic_eps = calculate_eps(net_income, shares_outstanding)
diluted_eps = calculate_eps(net_income, diluted_shares)

print(f"Basic EPS: ${basic_eps}")
print(f"Diluted EPS: ${diluted_eps}")
This code demonstrates our EPS calculation methodology. We compute both basic and diluted EPS to provide complete transparency. The diluted EPS accounts for stock options and convertible securities.
